### Week 2 Objectives

* Learn the fundamentals of networking infrastructure on Amazon Web Services.
* Practice launching and configuring Amazon EC2 instances.
* Understand AWS Identity and Access Management (IAM) Roles.
* Deploy a static website using Amazon S3.
* Create and manage a relational database using Amazon RDS MySQL.

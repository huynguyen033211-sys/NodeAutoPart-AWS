### Week 4 Objectives

* Learn how to implement load balancing and automatic scaling on AWS.
* Practice domain and DNS management with Amazon Route 53.
* Explore secure access to Amazon S3 using VPC Endpoints.
* Improve application security with AWS WAF.
* Manage sensitive credentials using AWS Secrets Manager.
